# Calculador de Interes

## Ejemplo totalmente funcional de una calculadora de interes mensuales de un prestamo en Dolares.

Este proyecto fue realizado como entregable durante el cursado de JavScript, en CoderHouse.
La funcionalidad de este proyecto permite:

* Ingresar datos para, con ellos, Calcular el Interes Mensual simulando un prestamo en USD.
* Mostrar, segun los datos ingresados, dicho Monto de Interes a pagar por mes segun su prestamo.

## Como utilizar el simulador:
* (1)- ingresar el monto del prestamo (en USD)
* (2)- Ingresar la tasa de interes del prestamo (en %)
* (3)- Ingresar la cantidad de meses del prestamo (max 300)
* (4)- La cantidad a pagar por mes aparecera automaticamente en su pantalla.
